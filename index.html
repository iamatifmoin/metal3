What is Metal3?

Metal3 is a BareMetal host provisioning service for Kubernetes. It is a self-hosted, self-managed, CNCF sandbox project and is also an infrastructure provider for the Cluster API. (K8S SIG life-cycle)

Metal3 in one line:

Bare metal host provisioning service for Kubernetes.

Why Metal3?

Reason 1

There are a number of great open-source tools for bare metal host provisioning, including Ironic. Metal3 aims to build on these technologies to provide a Kubernetes native API for managing bare metal hosts via a provisioning stack that is also running on Kubernetes.

Reason 2

The project is also building integration with the Kubernetes cluster-api project, allowing Metal3 to be used as an infrastructure backend for Machine objects from the Cluster API.

Reason 3

Metal3 is itself a Kubernetes application, so it runs on Kubernetes and uses Kubernetes resources and APIs as its interface.

Reason 4

The Metal project provides components for bare metal host management with Kubernetes. You can enroll your bare metal machines, and then, if you like, deploy Kubernetes clusters to them. From there, operating and upgrading your Kubernetes clusters can be handled by Metal3.
